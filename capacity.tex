\section{Executive Summary}
This analysis evaluates the hardware capacity for \textbf{1,000 concurrent MCU rooms} using \textbf{GStreamer} pipelines in MCU mode (real-time decoding, compositing, scaling, and encoding of multiple 720p streams into single 1080p outputs), with GPU acceleration (NVIDIA NVENC/CUDA) and lower DDR4 memory configurations.

\begin{itemize}
    \item 80\% (800 rooms): 6-participant rooms
    \item 20\% (200 rooms): 20-participant rooms
\end{itemize}

The proposed cluster consists of \textbf{40 nodes} (20 Small + 20 Big) and comfortably supports the workload with headroom.

\section{Conclusions and Recommendations}
The proposed 40-node heterogeneous cluster provides robust capacity for 1,000 concurrent GStreamer MCU rooms under GPU acceleration. Headroom supports maintenance, traffic spikes, and additional features (e.g., recording). 

Future work may include dynamic auto-scaling, detailed pipeline profiling with \texttt{gst-debug}, and power-efficiency analysis.
